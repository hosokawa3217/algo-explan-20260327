ジャンプ探索の全体像：具体例 data = 42

【アルゴリズムの要旨】

ジャンプ幅 m（√n）ごとに比較を行い、値を通り過ぎた瞬間にその直前の区間を戻りながら精査する。

例：n = 12, m = 4 の場合、インデックス 1 → 5 → 9 とジャンプして値を比較する。

幅 m=4 でジャンプ中、tbl[9] = 50 が data=42 を超えたため、その手前を精査。

実装

疑似言語プログラム解説

while ( i ≦ (n - m + 1) ) if (tbl[i] = data) return i elseif (tbl[i] > data) // 【１】戻り探索（dataを通り過ぎた区間） for (j を i-1 から i-m+1 まで 1減らす) if (tbl[j] = data) return j endfor return -1 endif i ← i + m endwhile // 【２】末尾探索（残った端の区間） for (j を i-m+1 から n まで 1増やす) if (tbl[j] = data) return j endfor

空欄【 １ 】

大きい値が見つかった位置 i の直前 i-1 から、一つ前のジャンプ地点の直後 i-m+1 までを「減らしながら」探します。

空欄【 ２ 】

ループを抜けた後、最後に比較した地点の直後 i-m+1 から、配列の本当の末尾 n までを「増やしながら」探します。
