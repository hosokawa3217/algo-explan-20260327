練習46：二分探索（外側境界型）

探索範囲の下限と上限の初期値に、実際の範囲の一つ外側を指定する特殊な実装パターンである。

low ← 0, high ← n + 1
mid ← (low + high) ÷ 2
while ( ( 【 1 】 が 1 より大きい ) 
       and ( tbl[mid] が data と等しくない ) )
  if ( tbl[mid] が data より小さい )
    low ← 【 2 】
  else
    high ← 【 3 】
  endif
  mid ← (low + high) ÷ 2
endwhile

練習46：アルゴリズムの要点

  • 境界の管理：low と high は常に「データが存在しないことが確定した領域の境界」を指し続ける。
  • 計算の簡略化：境界を外側に置くことで、更新時の +1 や -1 の処理を排除できる。
  • 終了条件：未探索要素がなくなった状態とは、high と low が隣り合わせになったときを意味する。
補足：ポインタが隣り合う（差が1になる）と、中身の箱がなくなります。

練習46：解答の導出

while ( ( high - low が 1 より大きい ) ...
  if ( tbl[mid] < data )
    low ← mid
  else
    high ← mid
  • 未探索範囲がある条件 ＝ high - low が 1 より大きいときである。
  • データが右側にある場合 ＝ low に mid を代入して範囲を狭める。
  • データが左側にある場合 ＝ high に mid を代入する。
解答： (1) カ high - low (2) ア mid (3) ア mid
