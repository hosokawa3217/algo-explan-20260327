テキスト T からパターン P を探す「力まかせ法」の空欄を埋めよ。

i ← 0, j ← 0
while ( i ＜ t_len and j ＜ p_len )
　　if ( T[i] = P[j] )
　　　　i ← i + 1, j ← j + 1
　　else
　　　　i ← 【　　 1 　　】
　　　　j ← 0
　　endif
endwhile
if ( 【　　 2 　　】 ) return i - j
else return -1

練習69：解説（バックトラック）

2/4

不一致が起きたとき、比較の開始位置を「1つ右」にずらす。

  • 開始位置の特定：今回の比較を開始したインデックスは i - j である。
  • ずらしの実行：不一致なら、開始位置の「1つ隣」から再スタートさせる。
  • 計算式：よって、テキストの指針 i を i - j + 1 に戻す。

練習69：解説（探索成功の判断）

3/4

ループが終わった後、どうやって「見つかった」と判定するか？

  • j の役割：j はパターンの「何文字目まで一致したか」を数えるカウンタ。
  • 成功の条件：j がパターンの長さ p_len と等しくなれば、全て一致した証拠。
  • 戻り値：一致した開始位置 i - j を返却する。

練習69：最終解答

4/4

文字列探索（力まかせ法）完成アルゴリズム

i ← 0, j ← 0
while ( i ＜ t_len and j ＜ p_len )
　　if ( T[i] = P[j] )
　　　　i ← i + 1, j ← j + 1
　　else
　　　　i ← i - j + 1
　　　　j ← 0
　　endif
endwhile
if ( j = p_len ) return i - j
else return -1

最終解答

【１】 ウ．i - j + 1
【２】 エ．j = p_len
