練習63：クイックソート（問題）

1/4

配列 A をクイックソートで整列させる。基準値を pivot としたとき、空欄を埋めよ。

○ quick_sort(整数型: left, right )
　　if ( left ＜ right )
　　　　p ← partition( left, right ) // 分割位置を決定
　　　　quick_sort( 【　　 1 　　】 ) // 左側を整列
　　　　quick_sort( 【　　 2 　　】 ) // 右側を整列
　　endif

練習63：解説（分割統治法）

2/4

クイックソートは、「分割して統治せよ」という考え方に基づいている。

  • 基準値（pivot）の選定：
    配列の中から一つの値を選び、それを基準にデータを二分する。
  • partition 関数：
    基準値より小さいものを左に、大きいものを右に集め、最終的な基準値の場所（添字 p）を返す。
  • 再帰呼び出し：
    分けられた左側の山と右側の山に対して、同じ手順をさらに適用していく。

練習63：解説（再帰の引数）

3/4

基準位置 p を使って、次に処理する範囲を指定しよう。

  • 左側の範囲（空欄1）：
    全体の左端 left から、基準位置の一つ手前 p - 1 までが対象。
  • 右側の範囲（空欄2）：
    基準位置の次 p + 1 から、全体の右端 right までが対象。
  • p の扱いに注意：
    基準位置にある要素は、この分割時点で既に「正しい場所」に配置されているため、次回の再帰に含める必要はない。

練習63：最終解答

4/4

クイックソート（quick_sort）完成コード

○ quick_sort(整数型: left, right )
　　if ( left ＜ right )
　　　　p ← partition( left, right )
　　　　quick_sort( left, p - 1 )
　　　　quick_sort( p + 1, right )
　　endif

最終解答（選択肢）

【１】 イ．left, p - 1 【２】 ア．p + 1, right
