疑似言語の詳細解説と論理構成

○ permutation(c, n, l, r)
if ( l == r ) // 並べ替えの注目位置が右端に達したとき
c[1] ~ c[n] を出力 // 1つの順列が完成したので出力する
else
for ( i を l から r まで ) // 未確定の範囲(l～r)から文字を選択
c[i] と c[l] を交換 // 選択したi番目の文字を、位置 l に固定する
permutation(c, n, l + 1, r) // 次の注目位置(l+1)以降を再帰的に生成
c[i] と c[l] を交換 // 次のループに備え、交換した文字を元の位置に戻す
endfor
endif

出口の論理

$l=r$ で再帰が止まり、結果が得られます。

文字の選択

交換により、候補を順番に先頭(l)へ据えます。

空欄【 1 】

隣の位置を処理するため、引数は l+1。解答：エ

バックトラック

元の状態に戻さないと、正しい順列が作れません。
