練習56：キューの操作（問題）

1/4

配列 Queue と変数 head, tail を用いてキューを実現する。データを追加する enqueue と、取り出す dequeue を完成させよ。

○ enqueue(整数型: data )
　　Queue[tail] ← data
　　tail ← 【　　 1 　　】

○ dequeue()
　　value ← Queue[head]
　　head ← 【　　 2 　　】
　　return value

練習56：解説（FIFOの仕組み）

2/4

キューは「先に入れたものが先に出る」という、順番待ちの列のような構造である。

  • head と tail の役割：
    head は「次に取り出すデータの位置」、tail は「次に追加するデータの位置」を指す。
  • enqueue（追加）の動き：
    現在 tail が指している場所にデータを入れ、tail を 1 つ進める。
  • dequeue（取出）の動き：
    現在 head が指している場所からデータを出し、head を 1 つ進める。

練習56：解説（ポインタの更新）

3/4

スタックとの違いは、追加と取り出しで「別の変数」が動く点にある。

  • enqueue：tail を更新：
    末尾に並ぶイメージ。追加するたびに末尾の位置を後ろへずらす。つまり tail ← tail + 1。
  • dequeue：head を更新：
    先頭がいなくなるイメージ。取り出すたびに先頭の位置を後ろへずらす。つまり head ← head + 1。
  • 注意点：
    どちらの操作もポインタは「増加」する方向へ動くのが基本である。

練習56：最終解答

4/4

キューの基本操作完成コード

○ enqueue(整数型: data )
　　Queue[tail] ← data
　　tail ← tail + 1

○ dequeue()
　　value ← Queue[head]
　　head ← head + 1
　　return value

最終解答（選択肢）

【１】 イ．tail + 1 【２】 ア．head + 1
