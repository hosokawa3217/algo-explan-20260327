練習62：基本挿入法（問題）

1/4

要素数 n の配列 A を昇順に整列させる基本挿入法のアルゴリズムである。空欄を埋めよ。

for ( i を 1 から n - 1 まで 1 ずつ増やす )
　　tmp ← A[i]
　　j ← i - 1
　　while ( ( j ≧ 0 ) and ( 【　　 1 　　】 ) )
　　　　A[j + 1] ← A[j]
　　　　j ← j - 1
　　endwhile
　　A[j + 1] ← 【　　 2 　　】
endfor

練習62：解説（挿入の仕組み）

2/4

トランプのカードを左手の中で並べ替えるときのような動きをするアルゴリズムである。

  • 注目する要素（tmp）：
    未整列部分の先頭要素 A[i] を取り出し、一時的に tmp に保管する。
  • 比較とシフト：
    整列済みの部分を後ろ（j 番目から 0 番目）から順に tmp と比較する。
  • 場所を空ける：
    tmp より大きい値があれば、それを一つ後ろ（j + 1 番目）にずらす。

練習62：解説（ループ条件と挿入）

3/4

while ループが止まるタイミングと、最後に tmp を戻す位置を考えよう。

  • 継続条件（空欄1）：
    整列済みの要素 A[j] が、挿入したい tmp よりも大きい間は、後ろへずらし続ける必要がある。
  • 挿入位置（空欄2）：
    ループを抜けたとき、j は「tmp 以下の値」または「配列の先頭 (-1)」を指している。そのため、その一つ後ろの j + 1 に tmp を入れる。
  • 効率性：
    既に整列されているデータに対しては、比較だけで済むため非常に高速。

練習62：最終解答

4/4

基本挿入法（insertion_sort）完成コード

for ( i を 1 から n - 1 まで 1 ずつ増やす )
　　tmp ← A[i]
　　j ← i - 1
　　while ( ( j ≧ 0 ) and ( A[j] ＞ tmp ) )
　　　　A[j + 1] ← A[j]
　　　　j ← j - 1
　　endwhile
　　A[j + 1] ← tmp
endfor

最終解答（選択肢）

【１】 イ．A[j] ＞ tmp 【２】 ア．tmp
