練習64：解説（分割と併合）

2/4

マージソートは、データを最小単位（1個）まで分けてから、再び戻していく手法である。

  • 再帰的な分割：
    中央 mid を基準に、配列をどんどん半分に分けていく。
  • 併合（Merge）：
    バラバラになった要素を「2つの整列済み配列を1つにまとめる」手法で合体させる。
  • 安定性：
    クイックソートとは異なり、同等な値の順序が崩れない「安定ソート」である。

練習64：解説（分割の境界線）

3/4

中央の添字 mid を使って、左右の範囲を正しく指定しよう。

  • 左半分の範囲（空欄1）：
    全体の左端 left から、中央 mid までを一つの範囲として分割する。
  • 右半分の範囲（空欄2）：
    中央の次 mid + 1 から、全体の右端 right までを範囲とする。
  • 終了条件：
    left ＜ right が成立しなくなったとき（要素が1つになったとき）、分割は停止する。

練習64：最終解答

4/4

マージソート（merge_sort）完成コード

○ merge_sort(整数型: left, right )
　　if ( left ＜ right )
　　　　mid ← ( left + right ) / 2
　　　　merge_sort( left, mid )
　　　　merge_sort( mid + 1, right )
　　　　merge( left, mid, right )
　　endif

最終解答（選択肢）

【１】 イ．left, mid 【２】 ア．mid + 1, right
