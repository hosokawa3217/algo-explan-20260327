練習57：解説（線形探索の基本）

2/4

線形探索（逐次探索）は、データの並びに頼らず端から順番に比較する手法である。

  • 探索の範囲：
    配列の最初の要素（添字0）から最後の要素（添字 n-1）までをもれなくチェックする。
  • 一致の判定：
    現在注目している要素 A[i] が、探している値 num と等しいかどうかを if 文で判定する。
  • 見つかった場合：
    その時点の添字 i を返して即座に終了する。

練習57：解説（ループと戻り値）

3/4

繰り返し条件の境界と、見つからなかった場合の処理に注意しよう。

  • ループの終点（空欄1）：
    要素数が n のとき、添字は 0 から n-1 までとなる。したがって、繰り返しは n-1 まで。
  • 比較対象（空欄2）：
    変数 i は添字を表す。中身を調べるには A[i] と記述する。
  • return -1 の意味：
    ループを最後まで回しても return i されなかった＝存在しない場合に、失敗を示す値を返す。

練習57：最終解答

4/4

線形探索（linearsearch）完成コード

○ linearsearch(整数型: num )
　　for ( i を 0 から n - 1 まで 1 ずつ増やす )
　　　　if ( A[i] == num )
　　　　　　return i
　　　　endif
　　endfor
　　return -1

最終解答（選択肢）

【１】 イ．n - 1 【２】 ア．A[i]
