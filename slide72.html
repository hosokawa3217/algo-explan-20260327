練習72：挿入ソート（問題）

1/4
整列済みの部分に新しい要素を「適切な位置」へ挿入するプログラムを完成させよ。
for ( i ← 1 to n - 1 )
　　tmp ← A[i]
　　j ← i - 1
　　while ( j ≧ 0 and 【　　 1 　　】 )
　　　　A[j + 1] ← A[j]
　　　　j ← j - 1
　　endwhile
　　A[j + 1] ← tmp
endfor

練習72：解説（要素の移動）

2/4
「ずらす」作業を繰り返すのはどのような時か？
  • 一時待避：挿入したい値（A[i]）を tmp に預け、その場所を空ける。
  • 右へ移動：tmp より大きな値がある限り、その値を右（j+1）へコピーして隙間を左へ広げていく。
  • 停止：tmp 以下の値にぶつかるか、左端（j < 0）に達したら移動を止める。

練習72：解説（継続の条件）

3/4
while文が「回り続ける」ための条件を考える。
  • 比較対象：挿入したい値 tmp と、整列済みの要素 A[j] を比較する。
  • 継続の判断：左側の A[j] の方が tmp より大きい間は、まだ挿入位置ではないため「ずらし」を続ける。
  • 条件式：A[j] ＞ tmp が成立する間、ループは継続する。

練習72：最終解答

4/4
挿入ソート（昇順）完成プログラム
for ( i ← 1 to n - 1 )
　　tmp ← A[i]
　　j ← i - 1
　　while ( j ≧ 0 and A[j] ＞ tmp )
　　　　A[j + 1] ← A[j]
　　　　j ← j - 1
　　endwhile
　　A[j + 1] ← tmp
endfor
