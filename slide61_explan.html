重ねて整列：ストゥージソート (Stooge Sort)

「2/3ずつ3回」呼び出す、ユニークな再帰構造
1
端っこをチェック
範囲の先頭 tbl[i] と末尾 tbl[j] を比較。
末尾の方が小さければ（逆順なら）入れ替える。
2
自分を呼び出す（3回）
要素数が3つ以上なら、全体の1/3の長さ t を計算。
次の「重なり」を持たせた順序で整列を行う。
1. stooge_sort(tbl, i, j - t) // 前2/3
2. stooge_sort(tbl, i + t, j) // 後2/3
3. stooge_sort(tbl, i, j - t) // 前2/3(再)
特徴
決して速くないが、再帰を理解するためのユニークな例。
重ねて整列することで、データが正しい位置へ動いていく。
「2/3」を3回重ねる再帰構造
核心：「前2/3 → 後2/3 → 前2/3」の順序で整列を重ね、全体を正しい並びへ収束させる。
