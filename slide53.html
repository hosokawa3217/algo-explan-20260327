二分探索木から値 num を探す関数 treesearch である。空欄 【 1 】 〜 【 4 】 を埋めよ。

○ treesearch(整数型: num )
　　node ← Root
　　while ( 【　　 1 　　】 )
　　　　if (node.data ＞ num )
　　　　　　【　　 2 　　】
　　　　else
　　　　　　【　　 3 　　】
　　　　endif
　　endwhile
　　if ( 【　　 4 　　】 )
　　　　"発見" を表示
　　else
　　　　"未登録" を表示
　　endif

練習53：解説（探索の仕組み）

2/4

二分探索木における効率的な探索の仕組みを理解しよう。

  • node ← Root：
    探索は常に根（一番上の節）からスタートする。
  • 探索を続ける条件：
    「まだ節が存在する」かつ「目的の値が見つかっていない」状態である。
  • 終了した後の判定：
    ループが終わったときに、node が空でなければ「見つかった」と判断できる。

練習53：解説（探索の継続と移動）

3/4

探索ループを制御する条件と、ポインタの更新ルールを整理しよう。

  • 探索を続ける条件（while）：
    「node が行き止まり（未定義）でない」かつ「目的の値が見つかっていない（node.data ≠ num）」間、下への移動を繰り返す。
  • ポインタの更新（移動）：
    「移動する」とは node の中身を書き換えることである。
  • (1) 左へ行く場合：
    node ← node.left を実行し、左の子に注目を移す。
  • (2) 右へ行く場合：
    node ← node.right を実行し、右の子に注目を移す。

練習53：最終解答

4/4

二分探索木からの探索（treesearch）完成コード

○ treesearch(整数型: num )
　　node ← Root
　　while ( (node が 未定義でない ) and
　　　　　　(node.data ≠ num ) )
　　　　if (node.data ＞ num )
　　　　　　node ← node.left
　　　　else
　　　　　　node ← node.right
　　　　endif
　　endwhile
　　if ( node が 未定義でない )
　　　　"発見" を表示

最終解答（選択肢）
