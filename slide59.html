ハッシュ関数 h(x) = x mod n を用いるハッシュ表（オープンアドレス法）の探索アルゴリズムである。空欄を埋めよ。

○ hash_search(整数型: num )
　　i ← 【　　 1 　　】
　　while ( H[i] != 空 )
　　　　if ( H[i] == num )
　　　　　　return i
　　　　endif
　　　　i ← 【　　 2 　　】
　　endwhile
　　return -1

練習59：解説（ハッシュ探索）

2/4

ハッシュ法は、データのキー値から直接「入っているはずの場所」を計算する。

  • 初期位置の決定：
    探索を開始する添字 i は、ハッシュ関数によって一意に決まる。
  • 衝突（コリジョン）の解決：
    もし計算した場所に別のデータが入っていた場合、オープンアドレス法では「次の場所」を順に調べていく。
  • 探索の終了：
    目的の値が見つかるか、あるいは「空」の場所に辿り着いたとき（そこにも無いことが確定）に終了する。

練習59：解説（ハッシュ関数と再ハッシュ）

3/4

空欄を埋めるための計算式を整理しよう。

  • 空欄1（初期添字）：
    ハッシュ関数 h(num) の戻り値をそのまま i に代入する。今回の関数は num mod n である。
  • 空欄2（次の添字）：
    線形探査法では添字を 1 増やすが、配列の末尾を超えないように n で割った余りをとる。
  • 式の意味：
    (i + 1) mod n とすることで、添字が n-1 の次は 0 に戻る円環構造を実現できる。

練習59：最終解答

4/4

ハッシュ法（hash_search）完成コード

○ hash_search(整数型: num )
　　i ← num mod n
　　while ( H[i] != 空 )
　　　　if ( H[i] == num )
　　　　　　return i
　　　　endif
　　　　i ← (i + 1) mod n
　　endwhile
　　return -1

最終解答（選択肢）

【１】 ア．num mod n 【２】 イ．(i + 1) mod n
