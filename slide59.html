挿入法 (Insertion Sort)

アルゴリズムの本質

整列済みの列に、未整列の値を「適切な位置を見つけて割り込ませる」手法です。

特徴まとめ

  • 計算量：O(n²)
  • トランプを並べる感覚に近い
  • 「ほぼ整列済み」のデータに非常に強い

挿入のイメージ

適切な位置(間)へ差し込む

ステップトレース

[10, 30, 20, 5] の整列 (i = 2 の場合)

① 比較の開始

変数 i は 2。
挿入したい要素は tbl[3] (値:20)。
これを左隣の tbl[2] (値:30) と比較します。

② 交換と確定

30 > 20 なので交換し、20は2番目へ。
次に tbl[1] (10) と比較。
10 < 20 なので、ここが定位置！(break)

実装の急所と j の制御

for (i を 1 から n - 1 まで増やす)
  /* 空欄1: 挿入要素の左隣から比較開始 */
  for (j を i から 1 まで 1 減らす)
    if (tbl[j] > tbl[j + 1])
      /* 隣り合う要素を交換 */
      wk ← tbl[j]
      tbl[j] ← tbl[j + 1]
      tbl[j + 1] ← wk
    else
      break /* 挿入位置確定 */
    endif
  endfor
endfor
論理的導出：
比較対象が tbl[j] と tbl[j + 1] であることが最大のヒント。
挿入したい要素は最初 i + 1 番目にあるので、
j + 1 = i + 1 つまり j = i から開始します。
