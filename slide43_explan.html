実装

疑似言語プログラム解説

while ( i ≦ (n - m + 1) ) if (tbl[i] = data) return i elseif (tbl[i] > data) // 【１】戻り探索 for (j を i-1 から i-m+1 まで 1減らす) if (tbl[j] = data) return j endfor return -1 endif i ← i + m endwhile // 【２】末尾探索 for (j を i-m+1 から n まで 1増やす) if (tbl[j] = data) return j endfor

解答：空欄【 １ 】

通り過ぎた現在地 i の直前 i-1 から、前回確認した位置の直後 i-m+1 までを「1ずつ減らしながら」探します。

解答：空欄【 ２ 】

最後に比較した地点の直後 i-m+1 から、配列の末尾 n までを「1ずつ増やしながら」探します。
