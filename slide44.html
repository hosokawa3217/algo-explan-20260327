練習４４：範囲内データの最小・最大

指定範囲 lv < tbl[ i ] < hv における最小値と最大値を求める。

大域: 整数型: lv ← 1000, hv ← 100000
○ min_and_max(整数型の配列: tbl, 整数型: n)
  整数型: i, min, max
  min ← [ 1 ]
  max ← [ 2 ]
  for ( i を 1 から n まで 1 ずつ増やす )
    if ( [ 3 ] )
      min ← tbl[ i ]
    endif
    if ( [ 4 ] )
      max ← tbl[ i ]
    endif
  endfor
  min と max の出力
lv ＜ tbl[ i ] ＜ hv

アルゴリズムの核心：初期値の逆転

  • 最小値 min には「これより大きい値はない」上限値 hv を最初に入れる。
  • 最大値 max には「これより小さい値はない」下限値 lv を最初に入れる。
  • 最初の比較で、範囲内のデータが必ず min や max に上書きされる。
  • 更新条件は tbl[ i ] が現在の暫定値を更新できるかどうかで判定する。
補足：最小値を探すときは最大値を、最大値を探すときは最小値を初期値にする。

練習４４：解答

各空欄の正解は以下の通りである。

  • [ 1 ]：最小値の初期値として hv を設定。
  • [ 2 ]：最大値の初期値として lv を設定。
  • [ 3 ]：tbl[ i ] が min より小さければ更新。
  • [ 4 ]：tbl[ i ] が max より大きければ更新。
(1) イ　　(2) ア
(3) ウ　　(4) ク
(1) hv　(2) lv　(3) tbl[ i ] < min　(4) tbl[ i ] > max
