練習61：基本交換法（問題）

1/4

要素数 n の配列 A を昇順に整列させる基本交換法のアルゴリズムである。末尾から順に比較する場合、空欄を埋めよ。

for ( i を 0 から n - 2 まで 1 ずつ増やす )
　　for ( j を 【　　 1 　　】 から i + 1 まで 1 ずつ減らす )
　　　　if ( A[j - 1] ＞ A[j] )
　　　　　　tmp ← A[j - 1]
　　　　　　A[j - 1] ← 【　　 2 　　】
　　　　　　A[j] ← tmp
　　　　endif
　　endfor
endfor

練習61：解説（隣り合う比較）

2/4

基本交換法（バブルソート）は、隣り合う要素の大小関係を正していく手法である。

  • i の役割：
    外側のループ i は、整列が完了する位置を示す。i=0 のときは、配列全体の最小値を 0 番目に持ってくる。
  • j の役割（内側ループ）：
    末尾から i の位置に向かって、隣り合う A[j-1] と A[j] を比較し、逆順なら入れ替える。
  • 泡のような動き：
    この操作により、最小値が先頭（i 番目）に向かって浮き上がってくる。

練習61：解説（探索範囲とスワップ）

3/4

末尾から i+1 まで戻るループの構成と、値の交換に注目しよう。

  • 空欄1（j の開始位置）：
    末尾から比較を開始するため、配列の最後の添字である n - 1 を指定する。
  • 空欄2（入れ替え）：
    A[j-1] の値を tmp に逃がした後、A[j-1] に A[j] の値を上書きする。
  • 境界の確認：
    j が i + 1 のとき、比較対象は A[i] と A[i+1] になり、正しく i 番目の値が確定する。

練習61：最終解答

4/4

基本交換法（bubble_sort）完成コード

for ( i を 0 から n - 2 まで 1 ずつ増やす )
　　for ( j を n - 1 から i + 1 まで 1 ずつ減らす )
　　　　if ( A[j - 1] ＞ A[j] )
　　　　　　tmp ← A[j - 1]
　　　　　　A[j - 1] ← A[j]
　　　　　　A[j] ← tmp
　　　　endif
　　endfor
endfor

最終解答（選択肢）

【１】 イ．n - 1 【２】 ア．A[j]
