練習６１：ストゥージソート（問題）

1/5
要素数が3以上のとき、範囲を3分割し、重なり合う2/3ずつを3回再帰的に整列せよ。 
○ stooge_sort(整数型の配列: tbl , 整数型: i , 整数型: j )
　　wk , t
　　if ( tbl[ i ] が tbl[ j ] より大きい )
　　　　wk ← tbl[ i ] ; tbl[ i ] ← tbl[ j ] ; tbl[ j ] ← wk
　　endif
　　if ( ( j - i + 1 ) が 3 以上 )
　　　　t ← ( j - i + 1 ) ÷ 3   /* 1/3の長さ */
　　　　stooge_sort( tbl , 【 １ 】 ) /* 先頭 2/3 */
　　　　stooge_sort( tbl , 【 ２ 】 ) /* 末尾 2/3 */
　　　　stooge_sort( tbl , 【 １ 】 ) /* 再度先頭 2/3 */
　　endif
選択肢： ア. i , j イ. i , i + t ウ. i + t , j エ. i , j + t オ. i - t , j カ. i , j - t

基本戦略と停止条件

2/5
  • 分割統治法：範囲を細かく分け、自分自身を呼び出す（再帰）。
  • 停止条件：要素数が 3 未満（j - i + 1 < 3）になれば終了。
  • 3段階の再帰：
    (1) 先頭 2/3
    → (2) 末尾 2/3
    → (3) 再度先頭 2/3 を処理。
ヒント：3分割したブロックの「2つ分」をどう指定するかが鍵。

1/3 の単位「t」の計算

3/5
  • t ← ( j - i + 1 ) ÷ 3
  • この式は、現在の整列範囲を3等分した時の「1ブロック分の長さ」を算出している。
  • 整数演算のため、端数は切り捨てられる。
  • この t を基準にして「どこまでが2/3か」を決定する。
範囲は [ i ] から [ j ] まで。全体の長さは j - i + 1。

空欄の導出：2/3 範囲の指定

4/5
  • 【 １ 】先頭 2/3 の範囲：
    全体の右端 j から 1/3分（t）を引いた地点まで。
    → 開始：i ／ 終了：j - t
  • 【 ２ 】末尾 2/3 の範囲：
    全体の左端 i から 1/3分（t）進んだ地点から最後。
    → 開始：i + t ／ 終了：j
中央の 1/3（i+t ～ j-t）は、どちらの再帰でも処理される「重なり」部分。

解答提示：ストゥージソート

5/5
○ stooge_sort(整数型の配列: tbl , 整数型: i , 整数型: j )
　　... (中略) ...
　　if ( ( j - i + 1 ) が 3 以上 )
　　　　t ← ( j - i + 1 ) ÷ 3
　　　　stooge_sort( tbl , i , j - t )
　　　　stooge_sort( tbl , i + t , j )
　　　　stooge_sort( tbl , i , j - t )
　　endif

最終解答（選択肢）

【 １ 】 カ．i , j - t
【 ２ 】 ウ．i + t , j
解答： (1)　カ　i , j - t　　(2)　ウ　i + t , j
