練習４７：単方向リストの探索（問題）

単方向リスト構造を用いた探索アルゴリズムである。空欄 【 1 】 に入る適切なコードを考えなさい。

listsearch(文字列型: moji)
ListTemplate: cr
cr ← ListHead
while ( ( cr が 未定義でない )
　　　and ( cr.data が moji と等しくない ) )
　　【  1  】
endwhile

練習４７：解説（探索の論理）

先頭 ListHead から順にポインタという「道しるべ」を辿り、目的の文字を探し出す。

探索を続ける条件は以下の2点である。

  • 1. cr がリストの末尾（未定義）に達していない。
  • 2. 現在のデータ cr.data が探している moji ではない。

これらを満たす間、「次の要素」へと注目を移す処理を繰り返す。

練習４７：解説（ポインタの更新）

「現在の要素」を「次の要素」に更新する具体的な方法を考えよう。

単方向リストにおいて、次の要素がどこにあるかは pointer に記録されている。

現在の cr を、今 cr が持っている pointer の値で上書きすることで、注目する場所を一歩進めることができる。

練習４７：解答提示

完成した探索アルゴリズムのプログラム

listsearch(文字列型: moji)
ListTemplate: cr
cr ← ListHead
while ( ( cr が 未定義でない )
　　　and ( cr.data が moji と等しくない ) )
　　cr ← cr.pointer
endwhile

最終解答

cr ← cr.pointer
