練習４９：単方向リストへの要素の追加

アルファベット順のリストに moji を追加する。単方向リストの prev と cr の間に新要素 tmp を追加する手順を考えよ。

// tmp：追加要素, cr：挿入位置, prev：直前要素
tmp.pointer ← cr
if ( prev が 未定義 )
　　【  1  】 // リストの先頭に追加
else
　　【  2  】 // 途中または末尾に追加
endif
選択肢： ア. cr.pointer ← tmp   イ. prev ← tmp   ウ. tmp ← ListHead   エ. ListHead ← tmp   オ. tmp.pointer ← prev   カ. prev.pointer ← tmp

練習４９：解説（繋ぎ替えの論理）

要素の挿入では、後続のデータを見失わないよう**「後ろを先に繋ぐ」**のが鉄則である。

1. まず tmp.pointer ← cr とし、新要素の後ろを確定させる。
2. 次に「前の要素」が tmp を指すように変更する。
3. このとき、**prev** が存在しない場合は「リストの起点」自体を書き換える必要がある。

練習４９：解答提示

完成したアルゴリズムは以下の通りである。

tmp.pointer ← cr
if ( prev が 未定義 )
　　ListHead ← tmp
else
　　prev.pointer ← tmp
endif

挿入位置が先頭か否かで、書き換えるポインタの対象を切り分けている点に注目せよ。

最終解答： (1)　エ　ListHead ← tmp　　(2)　カ　prev.pointer ← tmp
