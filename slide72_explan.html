概要

練習72：フォーマット変換とポインタ管理

処理の目的

文字列内の「tab」を、指定された文字数 n 個分の「空白（Δ）」に置き換えるフォーマット変換プログラムです。

ポインタ管理の分離

1文字が複数文字に変換されるため、入力元の添字 i と、出力先の添字 last はズレていきます。
そのため、「i とは独立して、last の位置を正確に更新し続ける」ことが重要になります。

出力先配列 b の「末尾がどこか」を常に last が指し示している状態をキープします。
核心

練習72：タブ発見時の反復処理

if ( a[ i ] が tab と等しい )
　for ( j を 1 から n まで 1 ずつ増やす )
　　last ← last + 1
　　b[ last ] ← "Δ"
　endfor
else
　last ← last + 1
　b[ last ] ← a[ i ]
endif
空欄の制御文
j を 1 から n まで
1 ずつ増やす

内側ループ（for文）の役割

タブが見つかった時、外側のループ（i）を一時停止させたまま、「指定された回数（n回）」だけ空白を連続で挿入します。その際も、1文字入れるごとに last を +1 して出力先を進めます。

「n 個の空白に置き換える」＝「n 回ループを回して Δ を代入する」と変換できるかが鍵です。
