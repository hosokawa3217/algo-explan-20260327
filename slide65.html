練習65：解説（ヒープの特性）

2/4

ヒープの特性「根が最大」を利用し、未整列の末尾と交換して確定させる。

  • 物理的な交換（Swap）：
    根（A[0]）にある最大値を、現在の未整列範囲の末尾（A[i]）へ移動し、値を固定する。
  • 再構築（Rebuild）：
    末尾から持ってきた「小さい値」が根に座るとヒープ構造が崩れるため、下方向へ沈めて再構成する。
  • 繰り返しの効果：
    これを行うごとに、配列の後ろから順に大きい値が確定していく。

練習65：解説（交換と範囲指定）

3/4

空欄に入る具体的な変数名と添字を特定しよう。

  • 空欄1（値の退避と交換）：
    最大値 A[0] を tmp に移した後、A[0] には現在の末尾である A[i] を代入する。
  • 空欄2（再ヒープ化の範囲）：
    末尾 i 番目は値が確定したため、ヒープから除外する。よって 0 番目から i - 1 番目までを再構築の対象とする。
  • 計算量：
    再構築は $O(\log n)$ で、これを $n$ 回繰り返すため全体で $O(n \log n)$ となる。

練習65：最終解答

4/4

ヒープソート（heap_sort）完成コード

○ heap_sort(整数型: n )
　　make_heap()
　　for ( i を n - 1 から 1 まで 1 ずつ減らす )
　　　　tmp ← A[0]
　　　　A[0] ← A[i]
　　　　A[i] ← tmp
　　　　rebuild_heap( 0, i - 1 )
　　endfor
