練習60：基本選択法（問題）

1/4

要素数 n の配列 A を昇順に整列させる基本選択法のアルゴリズムである。空欄を埋めよ。

for ( i を 0 から n - 2 まで 1 ずつ増やす )
　　min_idx ← i
　　for ( j を 【　　 1 　　】 から n - 1 まで 1 ずつ増やす )
　　　　if ( A[j] ＜ A[min_idx] )
　　　　　　min_idx ← j
　　　　endif
　　endfor
　　tmp ← A[i]
　　A[i] ← 【　　 2 　　】
　　A[min_idx] ← tmp
endfor

練習60：解説（最小値の選択）

2/4

選択法は、「未整列の部分から一番小さいものを探す」という動作を繰り返す。

  • i 番目の役割：
    外側のループの i は、現在「どの位置に入れる値を決めているか」を表す。
  • 最小値探し（j のループ）：
    i より後ろの要素を全て調べ、一番小さい値を持っている添字を min_idx に保存する。
  • 効率性：
    一回の交換で 1 つの要素の場所が確実に決まる。

練習60：解説（探索範囲と交換）

3/4

ループの開始点と、値の入れ替え処理を正確に把握しよう。

  • 空欄1（探索の開始）：
    i 番目の位置に入れる値を探すとき、比較するのは i の次の要素からである。したがって j は i + 1 から始まる。
  • 空欄2（交換の実行）：
    退避させておいた tmp を使い、A[i] に見つかった最小値 A[min_idx] を代入する。
  • アルゴリズムの終点：
    最後の 1 つ（n-1 番目）は自動的に決まるため、外側のループは n-2 までで良い。

練習60：最終解答

4/4

基本選択法（selection_sort）完成コード

for ( i を 0 から n - 2 まで 1 ずつ増やす )
　　min_idx ← i
　　for ( j を i + 1 から n - 1 まで 1 ずつ増やす )
　　　　if ( A[j] ＜ A[min_idx] )
　　　　　　min_idx ← j
　　　　endif
　　endfor
　　tmp ← A[i]
　　A[i] ← A[min_idx]
　　A[min_idx] ← tmp
endfor

最終解答（選択肢）

【１】 イ．i + 1 【２】 ア．A[min_idx]
