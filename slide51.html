二分探索木から目的の値 num を探すプログラムである。空欄を埋めなさい。

【 1 】 ← Root
while ( ( 【 1 】 が 未定義でない ) 
　　　and ( 【 1 】.data ≠ num ) )
　　if ( 【 1 】.data ＞ num )
　　　　【 1 】 ← 【 2 】
　　else
　　　　【 1 】 ← 【 3 】
　　endif
endwhile
if ( 【 4 】 )
　　表示("見つかりました")
endif

練習51：解説（探索の論理）

2/4

現在の値と num を比較することで、次に進むべき枝が確定する。

● node.data ＞ num の場合
　現在の値より num が小さいため、
　より小さい値が集まる 「左の子」 へ移動する。

● node.data ＜ num の場合
　現在の値より num が大きいため、
　より大きい値が集まる 「右の子」 へ移動する。

練習51：解説（ポインタの更新）

3/4

「移動する」とは、変数 node の中身を書き換えることである。

次に注目する節（node）を、今いる節の子へ上書きする。

(1) 左へ行く場合：
　　node ← node.left

(2) 右へ行く場合：
　　node ← node.right

練習51：最終解答

4/4

完成したプログラムと最終的な選択肢

node ← Root
while ( ( node が 未定義でない ) 
　　　and ( node.data ≠ num ) )
　　if ( node.data ＞ num )
　　　　node ← node.left
　　else
　　　　node ← node.right
　　endif
endwhile
if ( node が 未定義でない )
　　表示("見つかりました")
endif

最終解答（選択肢）

【１】　ア．node
【２】　ウ．node.left
【３】　エ．node.right
【４】　イ．node が 未定義でない
