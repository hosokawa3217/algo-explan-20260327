練習65：分割 (Divide) － 最小単位まで分解する

1. 中央の位置(中央 m)の計算
2. 偶数（6個）の分割
3. 1要素になるまで繰り返す
4. 要素数1 ＝ 整列済み
マージソートの「分割」は、中央の位置を境にデータを半分に分ける作業です。要素が1つになるまで分け続けることで、バラバラの状態（それ自体が整列済みである状態）を作ります。

練習65：準備 (Setup) － 比較用のテーブルを作る

1. 作業用配列 ll と rr
2. データのコピー（転記）
3. 番兵 hv (High Value)
4. 準備完了の配列
分割したデータを一度別の配列（llとrr）にコピーします。このとき、末尾に「番兵」と呼ばれる非常に大きな数（10000）を置くことで、どちらかの配列が空になってもエラーにならず比較を続けられます。

練習65：併合 (Merge) － 小さい順に拾い上げる

1. 先頭どうしの比較 (VS)
2. 小さい方を元の配列へ
3. 番兵との比較
4. すべて整列完了
左右の先頭を比べて「小さい方」を元の配列に戻します。どちらかが空になっても、末尾にある番兵（10000）が相手になるため、残ったデータが自然に順番通り吸い上げられていきます。
