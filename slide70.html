配列 A の要素を昇順に並べ替えるプログラムの空欄を埋めよ。
for ( i ← 0 to n - 2 )
　　for ( j ← 【　　 1 　　】 )
　　　　if ( A[j] ＞ A[j + 1] )
　　　　　　tmp ← A[j]
　　　　　　A[j] ← A[j + 1]
　　　　　　A[j + 1] ← tmp
　　　　endif
　　endfor
endfor

練習70：解説（ループの範囲）

2/4
内側のループ j はどこまで回せばよいか？
  • 確定の仕組み：1回のスキャンで、未確定部分の「最大値」が右端へ移動する。
  • 無駄を省く：外側ループ i 回目が終わるごとに、右側の i 個が確定済みとなる。
  • 比較対象：A[j] と A[j+1] を比べるため、j+1 が未確定の末尾に届けばよい。

練習70：解説（入れ替えの瞬間）

3/4
どのようなときに「交換」の処理を行うか？
  • 昇順のルール：小さい順に並べたいため、左側（j）が右側（j+1）より大きければ異常。
  • 判定の瞬間：A[j] ＞ A[j + 1] が成立したときのみ、中身を入れ替える。
  • データの待避：上書きで消えないよう、tmp 変数を使って 3ステップで交換する。

練習70：最終解答

4/4
バブルソート（基本）完成プログラム
for ( i ← 0 to n - 2 )
　　for ( j ← 0 to n - i - 2 )
　　　　if ( A[j] ＞ A[j + 1] )
　　　　　　tmp ← A[j]
　　　　　　A[j] ← A[j + 1]
　　　　　　A[j + 1] ← tmp
　　　　endif
　　endfor
endfor
